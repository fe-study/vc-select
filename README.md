<h1 align="center">@Vue Component - vc-select</h1>

## Install
`npm install vc-select --save`

## Dev

* Build

```node
npm run build

```

* Build dev

```node
npm run build:dev

```

* Build production 

```node
npm run build:prod

```

* Publish 

```node
npm run publish
```

* Build example

```node
npm run example
```

* run server

```node
npm run start
```
