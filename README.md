<h1 align="center">@Vue Component - vc-select</h1>

## Install

`npm install vc-select --save`

## Demo

[online demo](https://iwaimai-bi-fe.github.io/vc-select/examples/)

## Dev

* build

```node
npm run build

```

* build dev

```node
npm run build:dev

```

* build production 

```node
npm run build:prod

```

* build example

```node
npm run example
```

* run server

```node
npm run start
```

* publish 

```node
npm run release 
```
